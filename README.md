##  Kurulum ve Çalıştırma Talimatları

Projenin yerel sunucuda (localhost) sorunsuz çalışması için lütfen aşağıdaki adımları izleyin:

**1. Gerekli Paketlerin Kurulumu:**
Proje dizininde (server.js dosyasının bulunduğu klasörde) terminali açın ve gerekli Node.js modüllerini indirmek için şu komutu çalıştırın:
`npm install`(İlk yazılan çalışmazsa bunu deneyin)`npm install -y`

**2. Veritabanı (MongoDB) Bağlantısı:**
Bilgisayarınızda MongoDB Community Server'ın arka planda çalıştığından emin olun (Varsayılan port: 127.0.0.1:27017).

**3. Sunucuyu Başlatma:**
Terminal veya bash kısmına gidip aşağıda verilen kodu çalıştırın:
`node server.js`

**4. Tarayıcı Üzerinden Erişim:**
Sunucu çalıştıktan sonra web tarayıcınızı açın ve şu adrese gidin:
`http://localhost:3000`
